## 1. What is offered / Что предлагается

### 1.1. Anti‑Hallucination Core for LLMs

**EN**

A multilevel GRA-based **Anti‑Hallucination Core** that runs on top of your existing LLMs (OpenAI, local models, API).  
It takes multiple candidate answers (or multiple samples from one model), measures multiverse “foam” (inconsistency) between them and with external context, and returns a more consistent, fact‑aligned answer.

Typical integration points:

- RAG systems (question answering over documents).
- Chatbots with critical domains (finance, medicine, legal, support).
- Multi‑model ensembles (several LLMs in parallel).

Key benefits:

- Reduced hallucination rate without retraining your base model.
- Model‑agnostic: works with any provider / open‑source model.
- Extra diagnostics: per‑level consistency scores.

**RU**

Многоуровневое GRA‑ядро **Anti‑Hallucination**, работающее поверх ваших LLM (OpenAI, локальные модели, API).  
Оно принимает несколько вариантов ответов (или несколько сэмплов одной модели), измеряет мультиверсную «пену» (несогласованность) между ними и с внешним контекстом и возвращает более согласованный, фактуально выровненный ответ.

Типичные точки интеграции:

- RAG‑системы (вопрос–ответ по документам).
- Чат‑боты в критичных доменах (финансы, медицина, юриспруденция, поддержка).
- Ансамбли из нескольких LLM (несколько моделей параллельно).

Ключевые эффекты:

- Снижение уровня галлюцинаций без дообучения базовой модели.
- Независимость от конкретного провайдера/модели.
- Дополнительная диагностика: метрики согласованности по уровням.

### 1.2. Multiverse Optimizer for VPN / Network configs

**EN**

A multiverse meta‑optimizer for VPN / network configurations.  
Given a set of possible configurations (servers, protocols, routes, parameters), the optimizer searches for configurations with minimal “foam”: lower block probability, better latency and stability, subject to your constraints.

Use cases:

- VPN providers (white‑label optimization core).
- Corporate networks (multi‑hop routes, geo‑distribution).
- Resilient infrastructure under censorship or unstable connectivity.

**RU**

Мультиверсный мета‑оптимизатор VPN / сетевых конфигураций.  
Имея набор возможных конфигов (серверы, протоколы, маршруты, параметры), оптимизатор ищет конфигурации с минимальной «пеной»: меньшая вероятность блокировок, лучшая задержка и стабильность при заданных ограничениях.

Применение:

- VPN‑провайдеры (white‑label ядро оптимизации).
- Корпоративные сети (многозвеньевые маршруты, геораспределение).
- Устойчивые инфраструктуры в условиях цензуры и нестабильной связи.
